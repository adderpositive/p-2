# Test app

## Introduction
Demo of the application can be seen here: https://p-2-q8lh.vercel.app/. 

## First step

### Installation:

```
npm i
``` 

### Start localhost:
The application runs here http://localhost:3001/

```
npm run dev
```

### Build project with start
The application runs here http://localhost:3000/

```
npm run build
npm run start
``` 

### Test project
```
npm run test
```
